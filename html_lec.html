HTML Elements give Meaning to our Page Context

All elements have an opening and closing tag, except for the void elements.

What makes an element?

  • < > opening tag
  • </ >closing tag
  • Tags: an html tag is a set if characters enclosed in angle brackets that define an element's structure or behavior.
  • Element: An html element is made up of an opening tag, its content, and a closing tag

Block Level and Inline Elements

Block level elements usually strart in a new line and occupy the full width of their container.

Inline elements share their space with the parent block level element and only take up as much width as necessary. They also do not create a new line.

Void Elements

Void elements do not have closing tags. They only have opening tags.

HTML Attributes

HTML elements can contain additional data, known as attributes. Attributes are specified inside the opening tag for an element. They consist of a name, an = sign, and a value inside double quotes. For example:

<p name="example"> Some paragraph text goes here </p>

HTML for Page Layouts

HTML 5 offers new semantic elements that define the different parts of a layout.

  • <header> defines a header for a document section. they can be defined as <h1> through <h6>
  • <nav> defines a container for navigation links.
  • <article> defines an independent, self contained article.
  • <main> consists of content that is directly related to the central topic of a document.
  • <section> defines a section of the document.
  • <aside> defines content aside from the content (like a sidebar).
  • <footer> defines a foot for a document section
